# Project-Submission: DevOps Intern Task

## Overview
This project demonstrates secure server setup, containerized application deployment, 
container monitoring, user permission management, and firewall configuration.

## Tasks
- Task 1: Server Setup and SSH Configuration
- Task 2: Docker Installation and Application Deployment
- Task 3: Container Monitoring Script
- Task 4: Secure Monitoring Logs with Dedicated User
- Task 5: Firewall Configuration

## Tech Stack
- Ubuntu 24.04
- Docker
- Bash Scripting
- UFW Firewall
- Cron Jobs
- SSH
